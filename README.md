# Beverage Analysis Dashboard

## Project Overview
This dashboard analyzes beverage sales performance across different regions, brands, and time periods. 
It helps identify profitable regions, popular brands, and delivery efficiency.

## Tools Used
- Power BI
- Excel

## Key Metrics
- Total Sales
- Units Sold
- Average Units Sold per Region
- Average Delivery Time

## Key Insights
- Identified regions with the highest operating profit
- Analyzed monthly sales trends
- Compared beverage brands based on units sold
- Evaluated delivery performance across regions

## Dashboard Features
- Sales trend analysis
- Brand performance comparison
- Regional profit analysis
- Delivery efficiency insights
